Reference alone incomplete

While a developer’s guide should walk a developer through the basic usage of an API and its functionality, it can’t cover every possible use of that API in a coherent way. That is where articles and tutorials come in, to teach developers tangential or specialized uses of an API, like combining it with another service, framework, or API. —The Six Pillars of Complete Developer Documentation

Who writes the user guide?

Developers frequently write the reference content but almost never write the user guide ... because the user guide is much, much harder to write.

If the first tasks a learner masters are fundamentally similar to more complex tasks farther down the syllabus, the student begins to understand and apply those patterns. — Paul Gustafson, Lessons from a cephalopod
